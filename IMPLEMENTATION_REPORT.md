### 1. ✅ Создание модуля отслеживания (listening_tracker.py)

```
│  │ - Детальные записи   │  │ - Агрегированные данные     │ │
│  └──────────────────────┘  └─────────────────────────────┘ │
└─────────────────────────────────────────────────────────────┘
```

### Ключевые особенности

1. **Точность отслеживания**: Реальное время (wall clock), не зависит от скорости воспроизведения
2. **Производительность**: Запись в БД каждые 10 секунд, а не каждые 100ms
3. **Надежность**: Автоматическое завершение сессий при закрытии приложения
4. **Масштабируемость**: Индексы для быстрых запросов даже при большом объеме данных
5. **Гибкость**: Поддержка фильтрации по книгам, датам и периодам

## Примеры использования

### Программный доступ к статистике

```python
from database import DatabaseManager
from pathlib import Path

db = DatabaseManager(Path("data/audiobooks.db"))

# Получить статистику за сегодня
from datetime import datetime
today = datetime.now().strftime('%Y-%m-%d')
stats = db.get_daily_stats(start_date=today, end_date=today)

for stat in stats:
    print(f"{stat['audiobook_name']}: {stat['total_seconds']/3600:.2f} часов")

# Получить топ-5 книг за месяц
monthly = db.get_monthly_stats(year=2026, month=4)
top5 = sorted(monthly, key=lambda x: x['total_seconds'], reverse=True)[:5]

for book in top5:
    print(f"{book['audiobook_name']}: {book['total_seconds']/3600:.2f} часов")
```

### CLI просмотр статистики

```bash
# Общая сводка
python view_stats.py --summary

# Статистика за последние 7 дней
python view_stats.py --daily --days 7

# Топ-10 книг за месяц
python view_stats.py --top --period month --limit 10

# Годовая статистика
python view_stats.py --yearly
```
